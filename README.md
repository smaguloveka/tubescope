**TubeScope** — Красивый и мощный инструмент для глубокого анализа YouTube-конкурентов на базе YouTube Data API v3.

## Возможности

- Поиск и анализ каналов по названию, ссылке или @handle
- Полная статистика и обзор канала
- Графики публикаций (дни недели + время суток) и календарь
- Таблица видео с сортировкой и фильтрами
- Плейлисты с просмотром содержимого
- Топ видео по просмотрам / вовлечённости
- Контент-анализ (слова + теги)
- Сравнение до 3 каналов
- Экспорт в CSV / Excel / PDF / JSON
